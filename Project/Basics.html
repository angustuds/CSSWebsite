Selectors

Selectors from Interneting Is Hard

To choose which elements of your website you want to style you use selectors. These can either apply to everything of a certain type or just to a specific element.

The screenshot above shows the selector as p, so anything in the bracket will apply to all paragraph sections. You can have multiple selectors to apply style to several parts of the website as shown below. This would apply to all h1, h2 and h3:

To apply style to just a particular element you need to create classes. These then need to be called in the html file to show which elements you want to be styled by that class.

The class numbers is defined and then called using <div class="numbers">. At the end of the section that you want to style you must include </div> so it only styles the elements you want.

If you want some styling to only apply to a certain type of element, e.g. images, then you can create a class with the same name but that specifies that it is only for images. Text

To change the font type on your website you should use the font-family command. This should contain a list of font in case the chosen font isn't supported by the browser so there are fallback options.

You are also able to set the size, colour and style of your font. To change the font size you use font-size followed by the size in pixels. The font-weight property is used to change text to bold, the font-style changes it to italics and the text-decoration can do underline and line through.

To change the colour of text you use the color property. Most browsers support 140 colour names but you can use hex codes to set and colour you want. Below are examples of both methods to set the colour, firstly by name and then using a hex code.

Background

You can change the background of your website by either setting the colour or placing a background image. To change the colour you use the background-color property. You don't have to set the entire background to be one colour. You can also set it for only certain sections.
