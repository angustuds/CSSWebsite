Box Model

Box model from W3Schools

All html elements are considered as boxes. With CSS, every element is wrapped in a box with various attributes that can be changed. These are the margins, borders, padding, and the actual content.

Content

Whether it is text or images this is what the entire box is based around. When you change either the width or height property of an element, the changes happen to the content area so the total size of the element will be bigger if it has margins, borders or padding.

Padding

The padding is a clear area that surrounds the content increasing the size of the box and moving other boxes or a border further away. You can either set the padding to be the same all around or you can set the different sides to have different values or only set certain sides to have padding.

Borders

You can add a border to each element using the border property. There are three properties to a border; width, style and colour. The style has to be specified but if width and colour aren't then the width will be set to medium and the colour will be the same as the text. You can set all the properties using one single rule or in three separate rules. You do not have to have the same style, colour or width on every side of a border. If you define each property separately then you can list four different rules starting with the top border, followed by the right bottom, then the bottom border and finally the left border.

Each of the three images above have been styled with the border that they describe showing you the results of each rule and the different ways that you can set the border.

Margins

The margin is a transparent area which is placed around the outside of a border to move other elements further away. Like padding, you can set the margin size to be the same on every side or you can set the different sides to have different values or only set certain sides to have margins.

Background

With each box you can set the background to be a colour or an image. This will cover the content part of the box and the padding but not the border or margin. You can set this using either the background-color or the background-image command. If you use an image then you can also use background-position to set where you want the picture to be placed in the box and background-repeat to set whether you want the image to be repeated or not until it has covered the entire box.

Overflow

If you change the size of the box to be smaller than the default one then not all of the content will fit into the box, creating an overflow. There are several different ways of dealing with this using the overflow command. The default rule is that all the content will be visible and the content will be rendered outside of the element's box. You can change the overflow rule to hidden which will only allow as much content to be displayed as there is space inside the box. The two other options are scroll and auto. Both will not allow the content to go outside of its box however a scroll bar will appear to allow the user to view the rest of the content but in its box.
