By default every element has a position of static, meaning it is placed according to the natural flow of the page. Any element with static positioning cannot be moved by top, bottom, left or right rules.

Relative

Images from Interneting is Hard

If an element has a relative position then it is set based upon its normal position. If you change the top, bottom, left or right attribute of the element then the content will move but the box created for it will not move so other content will not move into the empty space. This also means that if you move it, it can be placed over other content as its box will still be in the same place.

The picture above shows how you would set the position to relative and then move the element to the right by 30 pixels. Although the "box" does not move with the element the area that is affected by border rules does move with the element.

Fixed

Image from Interneting is Hard

If you set the position to be fixed then the element will remain in the same position on the screen always, even if the user scrolls down. You use the top, bottom, left and right properties to set the position of the element relative to the edge of the browser window. To choose whether you want the fixed element to appear over or underneath other content you can use the z-index rule to set this.

Absolute

Image from Interneting is Hard

Absolute positioning is like relative but the element is positioned relative to the nearest positioned element rather than its original position. Also the box around the content moves with it so other content will move into its place if it moves. If there are no other positioned objects for it to position itself off then it will use the browser window.

This group of rules would change the position attribute of the element to absolute and then place it 80 pixels from the top of the webpage and on the right side.

Sticky

With sticky positioning, the element is position relative until it reaches a given position and then it becomes fixed. You must give at least one of top, bottom, left or right for sticky to know the position to become fixed. Note that sticky positioning is not supported on safari or internet explorer.
