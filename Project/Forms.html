Forms

Forms are how you can add an interface into your website for user interaction. The main purpose is to collect the users input and send it to a server. With forms you need to create the controls, provide a way for the user to submit their input and provide a URL to send the data to.

Basics

Firstly you need to use the <form> element. Then you need to define the input element of your form, this is very important as each part of your form must have a type. Later you will be shown the different types of input. The action attribute specifies where you want the users input to go when they submit it.

Every input field must have a name attribute associated with it. If there is no name then the data will not be sent.
